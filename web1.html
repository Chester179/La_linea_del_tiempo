Web 1.0

1990:

El Html y la web 1.0 Tim Berners-Lee define por fin el HTML, crea el primer navegador web, Viola WWW, que funcionaría en modo texto, y el uso de framesets o Marcos además de efectos como el parpadeo y las marquesinas, botones GIF, etc. Hacían más dinámica a la red.


1991:

Surgimiento de la Web 1.0 Consiste en un boche de documentos navegadores visuales como IE, web conectados por medio de hipervínculos, el usuario no podía interactuar.


1993:

La web y su uso gratuito para todo el público. La web 1.0 era estática y por lo general únicamente informativa. Usuario son solo lectores. Interacción reducida a formularios de contacto, inscripciones y boletines. Sitios direccionales y no colaborativos.
